Primefaces/Icon Showcase Icon Showcase

With the serenity themes you have several possibilities to insert an icon into your page.

  • FontAwesome 4.7
  • Material Icons
  • IvyIcons based on Streamline Icons

You can use them for example as follows:

FontAwesome


<i class="fa fa-home"></i>


<p:commandButton icon="fa fa-cog" value="Settings" />

Material Icons

home
<i class="material-icons">home</i>


<p:commandButton icon="ui-icon-settings" value="Settings" />

IvyIcons


<i class="icon ivyicon-house-1"></i>


<p:commandButton icon="icon ivyicon-cog" value="Settings" />

Please us a modern browser like Chrome, Firefox or Edge to browse the IvyIcons!
