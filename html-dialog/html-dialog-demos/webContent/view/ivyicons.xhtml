With the serenity themes you have several possibilities to insert an icon into your page.

  • FontAwesome 4.7
  • Material Icons
  • IvyIcons

IvyIcons are based on Streamline Icons and are only available with the serenity-ivy theme. You can use them for example as follows: 

        
 <i class="icon ivyicon-house-1"></i>  

 <p:commandButton icon="icon ivyicon-cog" value="Settings" />

Please us a modern browser like Chrome, Firefox or Edge to browse the IvyIcons!
