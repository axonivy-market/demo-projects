Primefaces/Icon Showcase Icon Showcase

With the serenity themes you have several possibilities to insert an icon into your page.

  • FontAwesome 6.1.0
  • Material Icons (Deprecated - Serenity only)
  • PrimeIcons
  • Streamline Icons Collection

You can use them for example as follows:

FontAwesome


<i class="fa-solid fa-house"></i>


<p:commandButton icon="fa-solid fa-gear" value="Settings" />

PrimeIcons


<i class="pi pi-home"></i>


<p:commandButton icon="pi pi-cog" value="Settings" />

Streamline Icons


<i class="si si-house-1"></i>


<p:commandButton icon="si si-cog" value="Settings" />

Please us a modern browser like Chrome, Firefox or Edge to browse the Streamline Icon Collection!
