Output/Resources Demo

This demo explains how Resources from the dialog resources folder, the webContent folder or the CMS can be used.

For using resources use the following code:

<h:outputStylesheet name="folder/style.css"/> Adds the stylesheet style.css to the HTML page. <h:outputScript name="folder/script.js"/> Adds the script script.js to the HTML page. <h:graphicImage name="folder/image.png"/> Adds the image image.png to the HTML page. \#{resource['folder/file.txt']} Adds the url of the file.txt to the HTML page.

Examples:

code result description <h:graphicImage name="images/dialog.png"/> If no library is defined then a resource is first searched within the dialog resources folder. If not found there then it is searched within the webcontent folder.
