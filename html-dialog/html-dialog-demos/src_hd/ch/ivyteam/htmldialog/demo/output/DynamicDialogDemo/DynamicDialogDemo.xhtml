Dynamic Dialog Demo

Dynamic dialogs allow you to open xhtml files in a new dialog.

  • This helps you to maintain the dialogs.
  • It's easier to test.
  • The view state is smaller and saves memory.
  • You can use the logic and the Html Dialog Data

The file has to be in the same package as the HTML Dialog.

Press the Button to open an Dynamic Dialog and write something in the Text box to see how it handles the Data
