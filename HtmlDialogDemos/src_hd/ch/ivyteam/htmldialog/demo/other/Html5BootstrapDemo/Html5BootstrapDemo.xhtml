Html 5 Bootstrap Demo

This demo shows a HtmlDialog that uses plain Html tags styled with Bootstrap instead of the PrimeFaces library.

  • This demo displays some of the capabilities of JSF 2.2 combined with the Bootstrap front-end framework.
  • Resize your browser window to witness the responsiveness of this page.
  • If you use this technique without JSF widgets you get almost full control of the Html delivered to the user, but you have to implement many things (like input validation) by yourself.
  • Have a look at the Html5Demo to get an explanation of the features.
